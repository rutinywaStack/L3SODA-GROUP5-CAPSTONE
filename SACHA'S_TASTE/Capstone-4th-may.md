# Capstone Project: Sacha’s Taste Website

## Project Overview

You will work in teams to design and build a **fully functional frontend website** for a Kigali-based food delivery business: **Sacha’s Taste**.

Link to sacha's Taste Instagram Page: <https://www.instagram.com/sachas_taste/>

The company currently operates only on Instagram and needs a professional website to showcase its services, menu, and brand.

Your goal is to **bring the provided Figma design to life using HTML & CSS**, with optional enhancements.

---

## Objectives

- Apply HTML & CSS skills to build a real-world website
- Collaborate effectively in teams using GitHub
- Translate a Figma design into a responsive website
- Present a working product through a live demo

---

## Technical Requirements

### Required

- Use **HTML5 & CSS3 only**
- Follow the provided **Figma design**
- Use **semantic HTML structure**
- Ensure **clean and readable code**
- Push all work to the **shared GitHub repository**
- Deploy the website using **GitHub Pages**

### Bonus Points (Optional)

- Add **JavaScript interactivity** (e.g., navbar toggle, sliders, animations)

---

## Team Structure

- Work in your assigned groups
- All contributions must be visible on GitHub (commits for each member)
- Teams should divide responsibilities (e.g., layout, styling, sections)

---

## Timeline (3 Days)

### Day 1 - 4th May

### Day 2 - 5th May
